Problem

Higher category theory is powerful but notoriously abstract. This project explores how to make concepts like Grothendieck fibrations, stacks, and layered ∞‑categories feel tangible by mapping them to motion, rhythm, and visually legible systems on the HTML5 Canvas.

Approach

The codebase is a small suite of interactive sketches. Each sketch uses layered geometry, orbiting “fiber” points, and color waves to evoke structures sitting over a base space. The combined gallery (gd.html) hosts three mini‑visualizations; there are standalone versions for deeper focus and an interactive playground with live controls.

Architecture

  • Canvas loops — Each sketch runs a requestAnimationFrame loop with lightweight draws for smooth 60fps.
  • Layers → nodes → fibers — Arrays model circular layers; each node spawns orbiting “fiber” points.
  • Color and trails — Trigonometric color waves and semi‑transparent strokes imply motion and depth.
  • Responsive sizing — Sketches resize to the viewport; the playground adds sliders and mouse warping.

Stack

  • HTML5 Canvas — Real‑time rendering with simple primitives (arcs, lines, fills).
  • Vanilla JavaScript — Animation state, geometry, and interaction without dependencies.
  • CSS (inline) — Minimal styling for contrast and legibility.

Sketches

  • Grothendieck Fibration Visualization (in gd.html): base points emit orbiting fibers connected back to the base line.
  • Grothendieck’s Cosmic Dance (gd2.html and in gd.html): multi‑layer rings of nodes with local fiber swirls.
  • ∞‑Stacks Spectral Swirl (gd4.html and in gd.html): fractal‑like bands with color‑shifted particles and occasional “meta‑lines.”
  • Playground (gd3.html): sliders for base count and fibers per base; mouse position warps orbits.

Challenges

  • Performance vs. richness — Balanced particle counts and draw order; used tiny primitives and alpha blends.
  • Clarity in abstraction — Tuned radii, speeds, and color phases so patterns read at a glance.
  • Responsiveness — Added resize handlers and parameter guards to keep behavior stable across viewports.

Outcomes

The combined gallery enables quick comparison between metaphors; the playground helps communicate how “fibers over a base” behave as parameters change. Everything is dependency‑free and easy to remix.
