Problem

Combining many PDFs into a single, correctly ordered document is a recurring admin task (e.g., reconciling ad invoices, receipts, or reports). Generic tools can be heavyweight or slow, and they often don’t support a quick review-and-reorder flow before merging.

Approach

Build a minimal Python CLI that scans an input/ folder, lists all PDFs, lets you interactively reorder them by number, and writes the result to output/merged_output.pdf. It relies on the pypdf library’s PdfWriter for a fast, dependency-light merge.

Architecture

  • Entry point: main.py orchestrates discovery → reorder → merge.
  • Discovery: Scans input/ for .pdf files, alphabetically sorted to start.
  • Reordering: Interactive prompt accepts comma/space‑separated numbers; validates count, range, and duplicates.
  • Merging: Uses pypdf.PdfWriter and writer.append(); writes to output/.
  • Folders: input/ (source PDFs), output/ (results); created if missing.
  • Error handling: Guardrails for missing folders, empty inputs, file not found, generic exceptions.

Stack

  • Python 3.12 — Simple, portable CLI scripting.
  • pypdf ≥ 5.4.0 — Reliable PDF append/merge via PdfWriter.
  • uv (optional) — Ultra‑fast Python package/dependency manager.

Usage

Place your PDFs in input/ and run the script. Two setup options:

With uv (recommended):

uv run python main.py

With pip + venv:

python -m venv .venv
source .venv/bin/activate   # Windows: .venv\\Scripts\\activate
pip install pypdf
python main.py

Follow the prompt to enter the desired order by number (e.g., 3, 1, 2), confirm, and the merged PDF will be saved to output/merged_output.pdf.

Challenges

  • Quick, safe reordering — Input parsing validates count, range, and prevents duplicates.
  • Cross‑platform paths — Uses os.path joins and binary read/write modes.
  • Minimal dependencies — Avoids heavyweight GUI stacks; keeps install surface tiny.

Outcomes

Streamlines a formerly manual process into a 10–30 second task. Common workflows include consolidating advertising invoices (e.g., Google Ads, LinkedIn, Reddit) and monthly statement packets. The tool’s interactive confirmation step reduces ordering mistakes before merging.

Project Structure
