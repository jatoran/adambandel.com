## Overview

A personal habit tracking system with streak analytics and visualization. Built to help maintain consistent habits through visual feedback and data-driven insights.

## Features

- Daily habit check-ins
- Streak tracking and analytics
- Visual calendar heatmaps
- Habit correlation analysis
- Export data to CSV
- Privacy-focused (local storage)

## Technical Stack

- **Frontend:** React, TypeScript
- **Visualization:** D3.js
- **Storage:** IndexedDB for local persistence
- **Styling:** Tailwind CSS

## Implementation

Uses React hooks for state management and D3.js for creating interactive visualizations. Data is stored locally in IndexedDB, ensuring privacy and offline capability. The analytics engine calculates streaks, averages, and correlations between different habits.
